POLLOCK

Generate placeholders for html wireframing without going near Photoshop.

Usage

Before you install the gem, you'll need ImageMagick. After that, it's just

$ sudo gem install pollock

You'll now have access to the web-app:

$ pollockserver

You're ready to start generating placeholders!

Syntax

/placeholders/width[xheight][-some_label_text][-color].format

Only a width is required to generate a square image. A label can be supplied to override the default behaviour of printing the image dimensions, and finally a #RRGGBB colour can be given to change the background.

The parameters are separated by '-' and any spaces in the label should be substitued with '_', sorry no non-alphanumeric characters yet.

You can supply any format extension ImageMagick is capable of generating.

http://localhost:1912/placeholders/100.jpg
http://localhost:1912/placeholders/200x100-with_some_text.jpg
http://localhost:1912/placeholders/305x100-with_a_colour-FF0000.png
http://localhost:1912/placeholders/305x100--FF0000.gif
